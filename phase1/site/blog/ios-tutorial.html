正文

iPhone用户有一个特别容易被忽略的扣费渠道——

App Store 订阅。

它和微信/支付宝完全独立，很多人换了几代手机都没检查过这里。😱

📍 路径：

设置 → 点顶部你的名字(Apple ID) → 订阅

进去后你会看到所有通过App Store订阅的服务——

⚠️ 这里面的坑特别大：

  • 有些App你可能已经删了好几年了
  • 有些是你点了"免费试用"然后忘取消的
  • 有些价格是美元标价，比想象中贵很多
  • 家庭共享可能让你在为别人的订阅付费

📋 检查清单（每项都要看）：

□ 有没有不认识/不记得的App？

□ 有没有已经删除但还在续费的App？

□ 有没有免费试用期过了但忘取消的？

□ 有没有价格单位是$而不是¥的？（汇率会让你多付）

□ 有没有比你记忆中贵的？（很多App悄悄涨价）

□ 年费订阅：到期后是按原价续还是活动价？

💡 进阶管理：

1️⃣ 家庭共享检查：

设置 → Apple ID → 家人共享 → 检查有没有共享的订阅在扣你的钱

2️⃣ 购买记录检查：

App Store → 右上角头像 → 购买记录 → 筛选"订阅"

可以看到完整的订阅扣费历史

3️⃣ 免费试用技巧：

开启试用后 → 马上去"订阅"页面取消 → 试用仍然有效直到到期 → 不会自动续费

今日作业：

现在打开 iPhone → 设置 → Apple ID → 订阅
