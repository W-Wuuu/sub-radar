微信的自动续费扣款入口藏得有点深，但关闭其实不难！

👇 跟着截图一步步来，30秒搞定 ✅

📍 路径：

微信 → 我 → 服务 → 钱包 → 支付设置 → 免密支付/自动扣款

（详细截图标注每一步点击位置）

进去之后你会看到一个列表：

📋 这里显示的是所有通过微信自动扣费的项目

  • 视频会员（腾讯视频/爱奇艺/优酷/B站...）
  • 音乐App（QQ音乐/网易云...）
  • 网盘（百度网盘/微云...）
  • 各种会员（京东PLUS/美团/饿了么...）
  • 甚至是某个你早已卸载的App！
点击任意一个项目 → 进入详情页 → 底部有"关闭服务"

⚠️ 注意：

• 关闭后当前周期仍有效，下个周期不再扣费

• 有的App会弹出挽留页面（"确定放弃权益吗？"）→ 点"仍要关闭"

• 关闭后建议截图留存，以防扯皮

💡 进阶技巧：

1️⃣ 定期检查：每月发工资后自查一次（设个日历提醒）

2️⃣ 开通即关：享受完首月优惠/免费试用后，马上关掉续费（权益仍保留到周期结束！）

3️⃣ 双重确认：关了之后下个月再检查一次，确认没有"复活"

常见问题：

Q：为什么我找不到"免密支付"？

A：更新微信到最新版本；有些旧版本叫"扣费服务"

Q：关闭后App说我会失去会员权益？

A：不会！当前已付费周期不受影响。只是下期不再自动续。

Q：已经扣了能退吗？

A：可以尝试。路径：微信支付消息 → 账单详情 → 联系商户 → 申请退款。成功率看平台。
