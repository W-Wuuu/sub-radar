## 一、微信支付

**路径**：
我 → 服务 → 钱包 → 支付设置 → 免密支付/自动扣款

**能看到什么**：
- 所有通过微信支付签约的自动扣费项目
- 包括：App会员、小程序订阅、视频号打赏自动充值、出行免密支付

**操作建议**：
1. 逐项查看，确认是否仍在使用
2. 不用的 → 点击进入 → "关闭服务"
3. 留用的 → 记下金额和续费日期
4. 每季度检查一次（设日历提醒）

**常见陷阱**：
- 小程序内开的会员通常也走微信支付，别漏了
- "视频号"相关的打赏/充值自动续费
- 快递柜/共享充电宝的免密支付也算

---

## 二、支付宝

**路径**：
我的 → 设置（右上角⚙️）→ 支付设置 → 免密支付/自动扣款

**额外检查**：
支付宝 → 我的 → 账单 → 右上角"分析"
可以看"文化休闲"类支出——大部分订阅都在这个分类。

**操作建议**：
1. 按"最近扣费"排序，优先关注活跃项
2. 搜索"会员""VIP""续费"→ 定位所有订阅类扣款
3. 不用的 → 解约
4. 设置月度消费限额提醒

**常见陷阱**：
- 淘宝/天猫的"先用后付"也算免密支付
- 支付宝的自动缴费（水电煤）也在这个入口管理

---

## 三、银行App

很多人不知道——银行App里也有一个自动扣费管理入口。

**路径（以招商银行为例）**：
首页 → 全部 → 支付管理 → 快捷支付/代扣协议

**其他银行**：搜索"代扣""快捷支付""签约管理"等关键词

**能看到什么**：
- 所有绑定该银行卡的快捷支付协议
- 包括一些不经过微信/支付宝的直连代扣

**操作建议**：
1. 检查有没有不认识/不记得的代扣协议
2. 不用的 → 解约
3. 建议只保留常用的2-3张卡的代扣

---

## 四、Apple App Store（iPhone用户）

**路径**：
设置 → 顶部你的名字(Apple ID) → 订阅

**操作建议**：
1. 逐项查看，注意价格单位（$ 还是 ¥）
2. 不用的 → 取消订阅
3. 免费试用中的 → 马上取消（试用仍有效）
4. 开启"续费提醒"（iOS会在扣费前发通知）
